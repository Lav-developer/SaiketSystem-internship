# SaiketSystem Internship Projects

This repository contains a collection of Java projects developed by **Lav Kush** during the **SaiketSystem** internship. Each task focuses on different fundamental and intermediate concepts of Java programming, including object-oriented programming (OOP), file handling, exception handling, and basic algorithms.

## 🚀 Projects Overview

### Task 1: Basic Calculator (`calc.java`)
A console-based calculator that performs fundamental arithmetic operations:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
Includes robust error handling for invalid inputs and division by zero.

### Task 2: Temperature Converter (`temp_conv.java`)
A simple utility to convert temperatures between Celsius and Fahrenheit:
- Celsius to Fahrenheit
- Fahrenheit to Celsius
Features a user-friendly menu and input validation.

### Task 3: To-Do List Application (`to_do.java`)
A command-line task management tool that allows users to:
- Add new tasks
- Mark tasks as complete
- View the current list of tasks with their completion status

### Task 4: Number Guessing Game (`number_guess.java`)
An interactive game where the computer generates a random number between 1 and 100, and the user has to guess it. The game provides hints ("Too high!" or "Too low!") and tracks the number of attempts.

### Task 5: Text File Analyzer (`analyzer.java`)
A file processing tool that reads a text file and provides:
- Total number of lines, words, and characters.
- A search functionality to find specific words (with optional case sensitivity) and count their occurrences.

### Task 6: Basic Banking System (`banking.java`)
A comprehensive banking application demonstrating OOP concepts (Inheritance, Abstraction, Polymorphism). Features include:
- Creating a savings account
- Depositing and withdrawing money (with insufficient funds handling)
- Viewing account balance and transaction history
- Applying interest to savings accounts

## 🛠️ Technologies Used
- **Language**: Java
- **Concepts**: Object-Oriented Programming (OOP), Exception Handling, File I/O, Data Structures (ArrayList)

## 💻 How to Run
1. Ensure you have the Java Development Kit (JDK) installed on your system.
2. Clone this repository or open the project directory in your terminal/command prompt.
3. Compile and run the desired Java file from the root directory. For example, to run the Number Guessing Game (Task 4):
   ```bash
   javac Task_4/number_guess.java
   java Task_4.number_guess
   ```
   *(Note: Task 1 does not use a package, so it can be compiled and run directly from the `Task_1` directory)*
